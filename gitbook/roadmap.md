# Roadmap

{% hint style="info" %}
See the note at the bottom -- this is a directional roadmap, not a commitment.
{% endhint %}

Where Monkey Finance is headed next:

1. **Banana Solana Vault** -- a vault for wrapping native SOL itself, not just SPL tokens like TKN, giving SOL holders the same wrap/stake mechanics available today.
2. **Community bTKN vaults** -- opening vault creation up beyond Monkey Finance's own token, so the community can spin up a wrap vault (and its own bTKN/LP staking pools) for other tokens they care about.
3. **Front-end updates** -- continued UI/UX improvements to the app as the protocol grows.

### MONKEY-aligned burn mechanics

Starting with future vaults, the burn share of each vault's wrap/unwrap fee won't burn that vault's own TKN -- it'll be used to burn MONKEY instead, while staking rewards keep paying out in the vault's own token as usual. That means every new vault added to the protocol feeds deflationary pressure directly into MONKEY, rather than each vault's burn only ever affecting its own token in isolation -- bullish for MONKEY as the protocol grows.

### Not a promise

This roadmap describes current thinking about where the protocol could go, not a commitment or guarantee. Timelines, priorities, and specific features may change based on market conditions, other protocol updates, community feedback, or anything else that comes up along the way -- nothing on this page should be treated as binding.
